# 如何開新案（操作步驟）

一份照著做就好的清單。給未來的自己與隊友 🙌

---

## 一次性：安裝 plugin（每位成員裝一次）

在 Claude Code 裡輸入：

```
/plugin marketplace add shellsirius/iniProj-plugin
/plugin install iniproj
```

裝完 `/exit` 重開 Claude Code 生效。之後「**開新專案**」這句話在**任何資料夾**都能用。

> 還沒推上 GitHub 前，可先用本機路徑測：
> `/plugin marketplace add /Users/shellsirius/00_websys/iniProj-plugin`

---

## 兩個核心概念

1. **「開新專案」是全域的** —— 在任何地方跟 Claude 說，它就幫你把整包範本長到你指定的位置。**你不用自己複製資料夾。**
2. **後續 skill 綁在新案資料夾裡** —— 分析 RFP（`importRFP`）、產 PRD、產 SA 這些，Claude 必須「**站進那個新案資料夾**」才讀得到。

> 💻 **macOS / Windows 都適用**：新手包網頁（`index.html`）任何系統雙擊都能開。指令（開網頁、讀 .docx）Claude 會**自動判斷你的作業系統**挑對版本。下面以 macOS 為例，Windows 邏輯相同。

---

## 步驟

### 1. 開新專案（生出一整包）
任何地方開 Claude Code，說：

```
開新專案
```

Claude 會問你三件事：**目標路徑**、**專案名稱**（建議 `proj-某某案`）、以及**售前文件在哪**。
- 把整包範本複製到指定位置——裡面已備好 PM／PRD／SA 的規則與工具。
- 你若給了**售前文件（RFP／服務建議書／合約）的路徑**，Claude 會順手把它們**複製進 `PM/input/`**（保留你的原檔），省得你手動拖。沒有就略過。

### 2. 站進新案資料夾，再啟動 Claude

> ⚠️ 順序：**先 cd（站進去），最後才 claude。** 這樣才讀得到這個案子自帶的 skill。

1. 打開**終端機**（⌘＋空白鍵，打「終端機」）
2. 輸入 `cd` 再**空一格**，把**新案資料夾**從 Finder **拖進終端機視窗** → 路徑自動填上 → 按 **Enter**
3. （想確認站對了：打 `pwd` 看結尾是不是新案名）
4. 輸入 `claude` 按 **Enter** → Claude 在這個資料夾啟動

### 3. 放入售前文件（若步驟 1 沒給才需要）
若開新專案時已給路徑、Claude 已搬進 `PM/input/`，**這步可略過**。
否則把 RFP／服務建議書／合約丟進 `PM/input/`（`.docx`、`.pdf`、`.txt/.md` 都可以，可一次放多份）。

### 4. 分析 RFP
跟 Claude 說：

```
importRFP
```

（「分析這份 RFP」也可以。）Claude 會：
- 讀 `PM/input/` 的文件
- 產出 `PM/output/gap-analysis.md`（合約已綁定 vs 待決策，附出處）
- 產出 `PM/prefill.js`（給新手包自動預填）
- **自動開啟 `PM/index.html`**（網頁會自己跳出來；若沒跳，手動雙擊它即可）

### 5. 補完問卷
在自動開啟的新手包網頁裡：
- 合約已綁定的題目**已自動帶出**，並標 📎 附合約依據
- 補完「待決策」的題目（例：無障礙、前端框架…）
- 按「下載 .md」

### 6. 收回產出
跟 Claude 說：

```
填好了
```

Claude 會把 `~/Downloads` 最新那份 .md 搬進 `PM/output/iniRFP.md`。
這份就是 PRD／SA／SRS 的上游依據（RFP 規格初稿，可能不完整）。

### 7. 往下游走
都在同一個新案資料夾裡，跟 Claude 說：
- `產PRD`（或 `iniRFPtoPRD`）→ 從 iniRFP 產 PRD R檔、更新索引與追溯表
- `會議產PRD`（或 `MeetingtoPRD`）→ 從 `PM/Meeting/` 的會議／訪談記錄產或修訂 PRD
- `產SA`（或 `PRDtoSA`）→ 把需求拆成功能、寫模組層 SA

## VS Code 路線（替代）

1. 先用「開新專案」生出新案資料夾（在任何已開的視窗說都行）
2. **檔案 → 開啟資料夾 → 選新案資料夾**
3. 開 Claude Code 擴充 → 說「importRFP」往下走

效果一樣（資料夾＝工作目錄）。

---

## 卡住時

- **說「開新專案」沒反應** → plugin 沒裝好或沒重啟。`/plugin install iniproj` 確認後重開 Claude Code。
- **說「importRFP」沒觸發** → 多半沒站進新案資料夾。`/exit` 離開，`cd` 到新案資料夾，再 `claude`。
- **網頁沒看到預填** → 確認 `PM/prefill.js` 有生成，且和 `PM/index.html` 在同一層。
- **下載的 .md 找不到** → 預設在 `~/Downloads`，跟 Claude 說「填好了」它會幫你搬。
